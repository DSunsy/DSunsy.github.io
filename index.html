Java学习

发表于 2016-11-02   |   分类于 Submit   |     |   阅读次数

眉间是银河，眼里有星辰

我打小就想做一个程序员，可却在岁月的成长里慢慢迷失曾经的憧憬。我很幸运，有人提醒了我，是姜铭提醒我要做一个优秀的人，是翁恺老师让我理解程序的宁静。陈同学则是优秀的竞争对手。

  • 数组的使用通常使用遍历操作，而操作通常使用For循环，对于for循环通常是从0到<数组的长度这样的方式。
  • 对于for循环的使用，也有第二种的方式；如：for(int k : data)表示对于data这个数组的每一个元素，循环中的每一轮都将其中的一个元素取出放到K中。它特别适合遍历整个数组时的使用。
  • 对于Break的使用，通常还有一种方式，即OUT: break OUT。
  • character有很多点的函数，包括isDight和isLowerCase。
  • 在java中==判断两个变量是不是同一个东西，而equals判断内容是否相同。对于比较还有有种是compareTo比较谁大谁小。大返回1，等于返回0，小于返回-1。
  • s.charAt（index）表示的是访问一个字符串中的单个字符，index代表的是字符串的索引。 从0到length-1。
  • substring（index）表示的是从当前索引位置到结束位置的子串。而substring（b,e）则代表的是从b索引位置到e索引位置之前的位置的子串。
  • 代码复制在程序中是质量不良的体现。
  • 函数是一块代码，它有零个或多个参数，做一件事情，返回零个或多个值。
  • java语言在调用语言的过程中永远只能把值传给函数。
  • 封装就是把数据和对数据的操作放在一起。
  • 如果有一个成员函数的名字和类的名字完全相同，则在创建这个类的每一个对象时候都会自动调用这个函数，这个函数叫做构造函数。
  • 如果一个类的成员函数中有多个和类名字相同的函数，并且这些函数的参数表互不相同，那么类在创建对象时候自动调用这样的构造函数的行为叫做重载。
  • private只能用于成员变量或者成员函数；所定义的私有成员变量或者成员函数只有在这个类的内部才能访问他。
  • 如果在一个类中的成员函数没有给定Public或者Private关键字，那么我们把他叫做friendly，和他位于同一个包的其他类可以访问。
  • 当一个编译单元里面有多个类 的时候，只有一个类可以是Public类型。而没有Public的类只能在这个包内部起作用，一旦出了这个包就失效。
  • in.next()表示读入一个单词，以空格作为标志。
  • in.nextline()表示读入一整行。

扶桑

发表于 2016-10-27   |   分类于 Stories   |     |   阅读次数

2016年10月10日的00：04分，我开始写这一篇自传式的故事。

那时我正在想着，我把我喜欢的女人拥入怀中，彼此感受着温暖，胸中竟真的暖了起来，我爱她，紧紧地抱着她，让她感受我爱的深沉，脑海的画面一转，原来我为了救她竟已经死了，她为我垂泪，我心中就涌起滔天巨浪，她是属于我的……可是她会不会变心，母亲会不会责怪她，我不禁这样想。画面再次变化，母亲看到了我留下的信，原谅了她，她依旧生活着。

也就在这时，我才意识到我还没有给我的家人，我的母亲留下自己的信。也是给自己的信。
我决定写一个震撼世人的故事，讲述自己是如何成长为一个“不自知”的怪物的。这篇故事从一个名叫大桑村的地方开始，也将必然在这里结束。

我是桑希，扶桑的桑；1995年8月19日，不知是夜亦或是昼，我降生在这个世界。我猜在产房外面，有初做人父手忙脚乱的父亲，透着傻气，那时他也许还是一个长不大的孩子呢；还有担心女儿安全其次期盼孙儿的焦虑的外婆，又或是，谁都没有。（鉴于当时的生活水平和环境，我完全可以假设母亲在一阵疼痛袭来之后准确预知我的到来，自行前往医院并独自生下了我，当然，这都是我的胡扯🌝）那一天到底发生了什么，都有些什么人在，亦或是说了些什么话，除了我母亲，谁也不知道。
总之，我出生辣。
